## 子代理分工原則（2026-03-31 確定）

### 三波分工法

```
第一波：數據 + 驗證（calc_verifier 把關）
第二波：分析 + 圖表
第三波：整合報告
```

### 每個數字必須經過驗證

```
1. 計算
2. 反算驗證（使用 calc_verifier.py）
3. 標記 ✅ 已驗證
```

### 大任務拆解原則
- 超過 5 個子任務 → 拆成多個子代理
- 子代理只做一件事
- 我（主要代理）負責收集 + 驗證 + 整合

### 計算驗證強制要求

所有投資相關計算都必須使用 calc_verifier.py 驗證：
- 張數 × 價格 = 金額
- 比例相加 = 100%
- 虧損金額和比例吻合
- 反算確認

### 用戶顯示偏好

- ETF/股票數量顯示為「1000股」而非「張」（1 張 = 1000 股）
