# 進階技術指標分析技能

> 基礎指標看趨勢，進階指標看轉折。

## 核心目的

RSI、MACD 只是基礎。這個技能提供 KDJ、OBV、ADX 等進階指標，幫你更精準地判斷轉折點和趨勢強度。

---

## 進階技術指標框架

### 第一層：進階指標清單

| 指標 | 計算 | 判讀 | 用途 |
|------|------|------|------|
| **KDJ** | 隨機指標 | K > 80 超買，K < 20 超賣 | 短期轉折 |
| **OBV** | 成交量指標 | OBV 創新高 = 量能確認 | 量能確認 |
| **ADX** | 趨勢強度 | ADX > 25 強趨勢，< 20 弱趨勢 | 趨勢強度 |
| **MACD 柱狀圖** | 動量指標 | 柱狀圖變小 = 動量衰退 | 動量衰退 |
| **布林帶寬度** | 波動率 | 寬度變小 = 即將突破 | 突破預判 |
| **乖離率** | 價格偏離均線 | 乖離率過大 = 回歸機會 | 回歸交易 |

### 第二層：進階指標組合

```
進場訊號組合：
1. KDJ 金叉（K > D）
2. OBV 創新高（量能確認）
3. ADX > 25（趨勢強）
4. MACD 柱狀圖變大（動量加強）
→ 四個條件都滿足 = 強進場訊號 🟢

出場訊號組合：
1. KDJ 死叉（K < D）
2. OBV 創新低（量能衰退）
3. ADX 下降（趨勢弱化）
4. MACD 柱狀圖變小（動量衰退）
→ 四個條件都滿足 = 強出場訊號 🔴
```

### 第三層：指標背離判斷

```
底背離 = 股價創新低但指標不創新低 → 可能底部 🟢
頂背離 = 股價創新高但指標不創新高 → 可能頂部 🔴

常見背離：
- 價格創新低，但 OBV 不創新低 = 底背離
- 價格創新高，但 RSI 不創新高 = 頂背離
- 價格創新高，但 MACD 柱狀圖變小 = 頂背離
```

## 執行流程

### 步驟 1：計算進階指標
- KDJ（14 日）
- OBV（成交量累計）
- ADX（14 日）
- MACD 柱狀圖
- 布林帶寬度
- 乖離率

### 步驟 2：判斷指標訊號
- 各指標當前位置
- 是否出現金叉/死叉
- 是否出現背離

### 步驟 3：組合判斷
- 多個指標是否同向
- 進場訊號強度
- 出場訊號強度

### 步驟 4：確認趨勢強度
- ADX 是否 > 25（強趨勢）
- 布林帶寬度是否變小（即將突破）
- OBV 是否創新高（量能確認）

### 步驟 5：得出結論
- 進場訊號評估
- 出場訊號評估
- 趨勢強度評估

## 常見組合策略

### 策略 1：KDJ + OBV 確認
```
進場條件：
1. KDJ 從超賣區（< 20）反彈
2. OBV 同時創新高
→ 量能確認的底部反彈
```

### 策略 2：ADX + 布林帶
```
進場條件：
1. ADX > 25（強趨勢）
2. 布林帶寬度變小（即將突破）
3. 股價在布林帶中線上方
→ 強趨勢中的突破
```

### 策略 3：背離 + 支撐位
```
進場條件：
1. 股價創新低
2. 指標不創新低（底背離）
3. 股價在支撐位反彈
→ 底背離 + 支撐位 = 強進場訊號
```

## 使用場景

1. **精準進場**：多個指標同向確認 = 進場信號強
2. **提前出場**：指標背離 = 可能頂部
3. **趨勢確認**：ADX > 25 = 趨勢強，可以持有
4. **風險預警**：指標齊聲轉弱 = 可能要減碼

---

## 常見誤區

❌ **誤區 1**：只看一個指標
✅ **正確**：多個指標組合判斷，信號更強。

❌ **誤區 2**：KDJ 超買就一定要賣
✅ **正確**：強趨勢中，KDJ 可以長期在超買區。

❌ **誤區 3**：背離就一定要反向操作
✅ **正確**：背離只是警示，要配合支撐壓力位確認。
