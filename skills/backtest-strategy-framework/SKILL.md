# 歷史回測策略框架技能

> 紙上談兵不如實戰驗證。

## 核心目的

再好的投資策略也要用歷史數據驗證。這個技能提供回測框架、績效指標計算、風險調整報酬分析，幫你判斷策略是否真的有效。

---

## 歷史回測框架

### 第一層：回測基礎

```
回測 = 用歷史數據模擬執行策略，看結果如何

步驟：
1. 定義策略規則（進場條件、出場條件、止損止利）
2. 用歷史數據模擬執行
3. 記錄每筆交易的進出價格、損益
4. 計算績效指標
5. 判斷策略是否有效
```

### 第二層：績效指標

| 指標 | 計算 | 判讀 | 重要性 |
|------|------|------|--------|
| **勝率** | 獲利交易數 / 總交易數 | > 50% 很好，> 60% 很優秀 | ⭐⭐⭐⭐ 強 |
| **平均獲利** | 平均每筆獲利金額 | 越高越好 | ⭐⭐⭐ 中等 |
| **平均虧損** | 平均每筆虧損金額 | 越低越好 | ⭐⭐⭐ 中等 |
| **獲利因子** | 總獲利 / 總虧損 | > 2.0 很好，> 1.5 可以 | ⭐⭐⭐⭐ 強 |
| **CAGR** | 年化複合成長率 | > 15% 很好，> 10% 可以 | ⭐⭐⭐⭐⭐ 最強 |
| **夏普比率** | (報酬率 - 無風險利率) / 波動率 | > 1.0 很好，> 0.5 可以 | ⭐⭐⭐⭐ 強 |
| **最大回撤** | 從高點到低點的最大跌幅 % | < 20% 很好，< 30% 可以 | ⭐⭐⭐⭐ 強 |
| **交易次數** | 總交易筆數 | 越多越好（樣本量大） | ⭐⭐⭐ 中等 |

### 第三層：風險調整指標

```
CAGR 只看報酬，不看風險
夏普比率 = 風險調整後的報酬

例子：
策略 A：CAGR 20%，波動率 30% → 夏普比率 0.67
策略 B：CAGR 15%，波動率 10% → 夏普比率 1.50

策略 B 雖然報酬低，但風險調整後更優秀
```

### 第四層：回測陷阱

```
過度擬合（Overfitting）：
- 策略在歷史數據上表現很好
- 但在未來數據上表現很差
- 原因：參數過度優化

防止方法：
1. 用樣本外數據驗證（Walk-forward test）
2. 不要過度優化參數
3. 保持策略簡單
4. 用多個時間段測試
```

## 執行流程

### 步驟 1：定義策略規則
- 進場條件（什麼時候買）
- 出場條件（什麼時候賣）
- 止損規則（虧損多少就停損）
- 止利規則（獲利多少就停利）
- 倉位管理（每次買多少）

### 步驟 2：收集歷史數據
- 股票代碼
- 時間段（至少 3-5 年）
- 日線數據（開盤、收盤、高低、成交量）

### 步驟 3：模擬執行
- 按照策略規則逐日執行
- 記錄每筆交易的進出價格
- 計算每筆交易的損益

### 步驟 4：計算績效指標
- 勝率、平均獲利、平均虧損
- 獲利因子、CAGR、夏普比率
- 最大回撤、交易次數

### 步驟 5：分析結果
- 策略是否有效（勝率 > 50%？）
- 風險是否可控（最大回撤 < 30%？）
- 報酬是否足夠（CAGR > 10%？）

### 步驟 6：優化策略
- 調整進場條件
- 調整止損止利
- 調整倉位管理

### 步驟 7：樣本外驗證
- 用新的時間段測試
- 看結果是否一致

## 常見策略回測

### 策略 1：均線交叉
```
進場：20 日均線 > 50 日均線（金叉）
出場：20 日均線 < 50 日均線（死叉）
止損：跌破 20 日均線 5%
```

### 策略 2：RSI 超賣反彈
```
進場：RSI < 30（超賣）+ 股價反彈
出場：RSI > 70（超買）
止損：跌破進場價 5%
```

### 策略 3：支撐位反彈
```
進場：股價在支撐位反彈
出場：股價突破壓力位
止損：跌破支撐位 5%
```

## 警示信號

🔴 **策略無效信號**：
- 勝率 < 50%（虧損交易多於獲利）
- 獲利因子 < 1.0（總虧損 > 總獲利）
- CAGR < 5%（報酬太低）
- 最大回撤 > 50%（風險太高）
- 樣本外驗證結果差（過度擬合）

🟡 **策略可疑信號**：
- 勝率 50-55%（邊界）
- 獲利因子 1.0-1.5（勉強）
- CAGR 5-10%（一般）
- 最大回撤 30-50%（中等風險）

🟢 **策略有效信號**：
- 勝率 > 55%（穩定獲利）
- 獲利因子 > 1.5（明顯獲利）
- CAGR > 10%（良好報酬）
- 最大回撤 < 30%（可控風險）
- 樣本外驗證結果一致（不過度擬合）

## 使用場景

1. **驗證策略**：新策略上線前必須回測
2. **優化策略**：用回測結果調整參數
3. **風險評估**：看最大回撤判斷風險
4. **績效對標**：用 CAGR 和夏普比率對標其他策略

---

## 常見誤區

❌ **誤區 1**：回測結果好就一定能賺錢
✅ **正確**：要考慮過度擬合、交易成本、滑點等因素。

❌ **誤區 2**：只看 CAGR，不看風險
✅ **正確**：要看夏普比率（風險調整後的報酬）。

❌ **誤區 3**：交易次數越多越好
✅ **正確**：交易次數多會增加交易成本和滑點。

❌ **誤區 4**：歷史回測結果就是未來結果
✅ **正確**：市場環境會變化，策略也要調整。
