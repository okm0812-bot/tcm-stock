# 現金流分析技能

> 現金流是企業的血液，EPS 可以造假，現金流騙不了人。

## 核心目的

EPS 可以通過會計技巧調整，但現金流是真實的。這個技能幫你深入分析公司的現金生成能力，判斷股息是否可持續、公司是否真的賺錢。

---

## 現金流分析框架

### 第一層：現金流三大組成

| 項目 | 說明 | 公式 |
|------|------|------|
| **營運現金流（OCF）** | 本業產生的現金 | 淨利 + 折舊攤銷 - 營運資金變化 |
| **投資現金流** | 購買資產、投資等支出 | 資本支出（CapEx）+ 投資支出 |
| **融資現金流** | 借貸、股利、股票發行等 | 股利支付 + 債務變化 |

### 第二層：自由現金流（FCF）

```
自由現金流 = 營運現金流 - 資本支出（CapEx）

意義：
- 正 FCF：公司有真實現金可以分配給股東
- 負 FCF：公司在燒錢，不可持續
- FCF > 淨利：公司質量好（現金比利潤更真實）
```

### 第三層：現金流質量指標

| 指標 | 計算 | 判讀 |
|------|------|------|
| **FCF 殖利率** | FCF / 市值 | > 5% 很好，> 3% 可以，< 1% 要小心 |
| **FCF 轉換率** | FCF / 淨利 | > 100% 很好，50-100% 正常，< 50% 要注意 |
| **股息覆蓋率** | FCF / 股利支付 | > 1.5x 很安全，1-1.5x 正常，< 1x 不可持續 |
| **CapEx / 營收** | 資本支出 / 營收 | 判斷公司投資強度 |

## 執行流程

### 步驟 1：收集現金流數據
- 營運現金流（OCF）
- 資本支出（CapEx）
- 自由現金流（FCF）
- 股利支付金額
- 過去 3-5 年歷史數據

### 步驟 2：計算 FCF 相關指標
- 自由現金流 = OCF - CapEx
- FCF 殖利率 = FCF / 市值
- 股息覆蓋率 = FCF / 股利支付

### 步驟 3：分析現金流趨勢
- FCF 是否穩定增長？
- CapEx 是否合理？
- 股利是否可持續？

### 步驟 4：對比 EPS vs FCF
- EPS 和 FCF 是否同向？
- 如果 EPS 高但 FCF 低，要警惕
- 如果 FCF 高但 EPS 低，可能是好機會

### 步驟 5：得出結論
- 公司現金生成能力評估
- 股息可持續性評估
- 投資風險評估

## 關鍵警示信號

🔴 **高風險信號**：
- FCF 持續為負
- FCF < 淨利的 50%（現金質量差）
- 股息覆蓋率 < 1x（股息不可持續）
- CapEx 急劇上升（可能面臨產業變革）

🟡 **中等風險信號**：
- FCF 波動大
- FCF 轉換率 50-70%
- 股息覆蓋率 1-1.5x

🟢 **健康信號**：
- FCF 穩定增長
- FCF 轉換率 > 100%
- 股息覆蓋率 > 1.5x
- FCF 殖利率 > 3%

---

## 使用場景

1. **股息投資**：買之前先檢查股息是否可持續
2. **持股檢視**：定期檢查現金流是否惡化
3. **風險預警**：現金流惡化通常是股價下跌的先兆
4. **估值調整**：FCF 比 EPS 更能反映真實價值

---

## 注意事項

- **會計差異**：不同國家的會計準則可能有差異
- **季節性**：某些行業有明顯季節性，需要年度對比
- **一次性項目**：某些年份可能有大額投資或資產出售，需要調整
- **數據滯後**：現金流數據通常滯後 1-3 個月
