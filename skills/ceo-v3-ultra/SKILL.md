## 執行流程（Ultra 版）

### 步驟 1：抓取即時數據
使用 `stock_analysis.py` 抓取：
- 現價、PE、EPS、ROE、殖利率、52週高低
- 自由現金流（FCF）
- 流通股數

使用 `var_risk_analysis.py` 計算：
- VaR (95%、99%)
- 夏普比率
- 最大回撤
- 年化波動率
- 日勝率

使用 `scenario_dcf.py` 計算：
- 三情境 DCF（樂觀/中性/悲觀）
- 安全邊際
- 合理價值區間

### 步驟 2：計算持股 Quality Score
根據以下指標評分：
- ROE > 10% = +2分
- ROE 5-10% = +1分
- ROE < 5% = +0分
- PE < 15x = +2分
- PE 15-25x = +1分
- PE > 25x = +0分
- 安全邊際 > 30% = +3分
- 安全邊際 0-30% = +1分
- 安全邊際 < 0% = +0分
- 現金流為正 = +2分
- 產業前景好 = +1分

 Quality Score = ROE分 + PE分 + 安全邊際分 + 現金流分 + 產業分

| Quality Score | 等級 | 建議 |
|--------------|------|------|
| 8-10 | 🏆 A級 | 核心持有，可加碼 |
| 5-7 | B級 | 持有，觀望 |
| 3-4 | C級 | 減碼，考慮換股 |
| 0-2 | D級 | 停損，觀望 |

## 數據解讀指南

### Quality Score 怎麼看

| 分數 | 解讀 |
|------|------|
| 8-10 分 | 這檔股票是「高品質」，值得長期持有或加碼 |
| 5-7 分 | 這檔股票「還可以」，持有但不急著加碼 |
| 3-4 分 | 這檔股票「有問題」，考慮慢慢減碼 |
| 0-2 分 | 這檔股票「爛」，果斷停損 |

### VaR 怎麼看

VaR (95%) = -3%
→ 如果持有 100 萬，有 5% 的機率單日會虧損超過 3 萬

### DCF 安全邊際怎麼看

安全邊際 = 40%
→ 股票內在價值是 100 元，現在股價 60 元
→ 你用 60 元買到價值 100 元的東西，40% 是「送的」
