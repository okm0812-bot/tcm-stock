# 同業對標分析技能

> 知己知彼，百戰不殆。

## 核心目的

在投資決策前，不能只看目標公司本身，還要看它在同業中的相對位置。這個技能幫你快速對標競爭對手，找出目標公司的優勢和劣勢。

---

## 對標框架

### 第一層：基本面對標

| 指標 | 說明 | 為什麼重要 |
|------|------|----------|
| **營收規模** | 年營收、年增率 | 判斷市場地位 |
| **毛利率** | 毛利 / 營收 | 判斷定價權 |
| **營業利益率** | 營業淨利 / 營收 | 判斷本業效率 |
| **淨利率** | 淨利 / 營收 | 判斷整體獲利 |
| **ROE** | 淨利 / 股東權益 | 判斷資本效率 |
| **EPS** | 每股盈餘 | 判斷每股獲利 |

### 第二層：成長性對標

| 指標 | 說明 | 為什麼重要 |
|------|------|----------|
| **營收成長率** | 年營收增長 % | 判斷成長動能 |
| **淨利成長率** | 年淨利增長 % | 判斷獲利加速 |
| **EPS 成長率** | 年 EPS 增長 % | 判斷每股成長 |

### 第三層：估值對標

| 指標 | 說明 | 為什麼重要 |
|------|------|----------|
| **本益比（PE）** | 股價 / EPS | 判斷估值貴便 |
| **PEG 比率** | PE / 成長率 | 判斷成長性估值 |
| **EV/EBITDA** | 企業價值 / 息稅前利潤 | 判斷企業估值 |
| **股息殖利率** | 股息 / 股價 | 判斷現金回報 |

### 第四層：現金流對標

| 指標 | 說明 | 為什麼重要 |
|------|------|----------|
| **自由現金流（FCF）** | 營運現金流 - 資本支出 | 判斷真實現金生成 |
| **FCF 殖利率** | FCF / 市值 | 判斷現金回報率 |
| **營運現金流** | 本業產生的現金 | 判斷現金質量 |

## 執行流程

### 步驟 1：確定對標對象
- 目標公司：鴻海（2317）
- 競爭對手：台積電、聯發科、日月光、Flex、Jaco 等
- 選擇 3-5 家最相關的對手

### 步驟 2：收集財務數據
- 最新年度財報（2025年）
- 最新季度財報（2026年 Q1）
- 過去 3 年歷史數據（判斷趨勢）

### 步驟 3：製作對標表格
- 橫軸：各公司
- 縱軸：各項指標
- 用顏色標記：綠色（領先）、黃色（中等）、紅色（落後）

### 步驟 4：分析差異原因
- 為什麼 A 公司毛利率高？
- 為什麼 B 公司 ROE 低？
- 差異是結構性還是週期性？

### 步驟 5：得出結論
- 目標公司的相對優勢
- 目標公司的相對劣勢
- 估值是否合理

## 使用場景

1. **新股票分析**：買之前先對標，看它在同業中的位置
2. **持股檢視**：定期對標，看是否還值得持有
3. **加碼決策**：對標後發現相對便宜，才考慮加碼
4. **止損判斷**：對標後發現相對落後，考慮減碼或止損

---

## 注意事項

- **數據時效性**：財報數據可能滯後 1-3 個月，需定期更新
- **行業差異**：不同行業的指標基準不同，對標時要考慮行業特性
- **週期性**：某些公司可能處於週期頂部或底部，需要歷史對比
- **一次性項目**：某些公司可能有一次性收入或支出，需要調整
