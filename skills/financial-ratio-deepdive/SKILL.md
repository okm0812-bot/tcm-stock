# 財務比率深化分析技能

> 表面看 EPS，深度看杜邦。

## 核心目的

EPS 只是表面，真正的財務健康要看杜邦分析、現金流比率、效率比率。這個技能提供深度財務分析框架，幫你看穿財務報表。

---

## 財務比率深化框架

### 第一層：杜邦分析（ROE 分解）

```
ROE = 淨利率 × 資產周轉率 × 權益乘數

意義：
- 淨利率：賺錢能力
- 資產周轉率：資產利用效率
- 權益乘數：財務槓桿

例子：
ROE 15% = 10% × 1.5 × 1.0（低槓桿，靠本業賺錢）
ROE 15% = 5% × 1.0 × 3.0（高槓桿，靠借錢放大）
→ 前者更健康
```

### 第二層：現金流比率

| 比率 | 計算 | 判讀 | 警示 |
|------|------|------|------|
| **營運現金流 / 淨利** | OCF / 淨利 | > 100% 很好，50-100% 正常，< 50% 要警惕 | < 50% = 現金質量差 |
| **自由現金流 / 淨利** | FCF / 淨利 | > 80% 很好，50-80% 正常，< 50% 要警惕 | < 50% = 投資支出過大 |
| **營運現金流 / 流動負債** | OCF / 流動負債 | > 1.0 很好，0.5-1.0 正常，< 0.5 要警惕 | < 0.5 = 短期償債能力弱 |
| **股息覆蓋率** | FCF / 股利支付 | > 1.5x 很安全，1-1.5x 正常，< 1x 不可持續 | < 1x = 股息不可持續 |

### 第三層：效率比率

| 比率 | 計算 | 判讀 | 用途 |
|------|------|------|------|
| **資產周轉率** | 營收 / 平均資產 | 越高越好，判斷資產利用效率 | 對比同業 |
| **應收帳款周轉率** | 營收 / 應收帳款 | 越高越好，判斷收款速度 | 判斷營運效率 |
| **存貨周轉率** | 營收 / 存貨 | 越高越好，判斷存貨流動速度 | 判斷庫存風險 |
| **應付帳款周轉率** | 營收 / 應付帳款 | 越低越好，判斷付款速度 | 判斷現金流管理 |

### 第四層：償債能力比率

| 比率 | 計算 | 判讀 | 警示 |
|------|------|------|------|
| **負債比率** | 總負債 / 總資產 | < 50% 很好，50-70% 正常，> 70% 要警惕 | > 70% = 財務風險高 |
| **利息覆蓋率** | 營業淨利 / 利息支出 | > 5x 很好，3-5x 正常，< 3x 要警惕 | < 3x = 償債能力弱 |
| **債務 / EBITDA** | 總負債 / EBITDA | < 2x 很好，2-3x 正常，> 3x 要警惕 | > 3x = 槓桿過高 |

### 第五層：多年度對比

```
單年財務比率 = 靜態快照
多年度對比 = 動態趨勢

重點看：
1. 淨利率是否穩定或上升？
2. ROE 是否穩定或上升？
3. 現金流比率是否穩定或上升？
4. 負債比率是否穩定或下降？
5. 效率比率是否穩定或上升？

趨勢向好 = 公司在改善
趨勢向壞 = 公司在惡化
```

## 執行流程

### 步驟 1：收集財務數據
- 過去 3-5 年的財務報表
- 淨利、營運現金流、自由現金流
- 資產、負債、股東權益
- 應收帳款、存貨、應付帳款

### 步驟 2：計算杜邦分析
- 淨利率 = 淨利 / 營收
- 資產周轉率 = 營收 / 平均資產
- 權益乘數 = 平均資產 / 平均股東權益
- ROE = 淨利率 × 資產周轉率 × 權益乘數

### 步驟 3：計算現金流比率
- 營運現金流 / 淨利
- 自由現金流 / 淨利
- 股息覆蓋率 = FCF / 股利支付

### 步驟 4：計算效率比率
- 資產周轉率
- 應收帳款周轉率
- 存貨周轉率
- 應付帳款周轉率

### 步驟 5：計算償債能力
- 負債比率
- 利息覆蓋率
- 債務 / EBITDA

### 步驟 6：多年度對比
- 繪製趨勢圖
- 判斷改善或惡化

### 步驟 7：得出結論
- 財務健康度評估
- 風險警示
- 改善或惡化趨勢

## 警示信號

🔴 **高風險信號**：
- 淨利率持續下降
- 現金流比率 < 50%（現金質量差）
- 負債比率 > 70%（槓桿過高）
- 利息覆蓋率 < 3x（償債能力弱）
- 股息覆蓋率 < 1x（股息不可持續）

🟡 **中等風險信號**：
- 淨利率波動大
- 現金流比率 50-70%
- 負債比率 60-70%
- 效率比率下降

🟢 **健康信號**：
- 淨利率穩定或上升
- 現金流比率 > 100%
- 負債比率 < 50%
- 利息覆蓋率 > 5x
- 效率比率穩定或上升

---

## 使用場景

1. **財務健康檢查**：定期檢查杜邦分析和現金流比率
2. **風險預警**：現金流比率惡化 = 可能要減碼
3. **估值調整**：現金流質量差 = 應該用更低的 PE 倍數
4. **對標分析**：用杜邦分析對比同業，看誰更健康

---

## 常見誤區

❌ **誤區 1**：只看 ROE，不看杜邦分解
✅ **正確**：ROE 15% 可能是靠槓桿，也可能是靠本業。要看分解。

❌ **誤區 2**：EPS 高就是好公司
✅ **正確**：要看現金流比率。EPS 高但現金流低 = 現金質量差。

❌ **誤區 3**：負債比率越低越好
✅ **正確**：適度槓桿可以放大 ROE。關鍵是利息覆蓋率是否足夠。

❌ **誤區 4**：只看單年數據
✅ **正確**：要看多年度趨勢。單年好不代表持續好。
