# Plan CEO Review - CEO Perspective Planning

Strategic planning skill from a CEO/founder perspective.

## When to Use This Skill

Use this skill when the user says:

- "think bigger"
- "expand scope"
- "strategic review"
- "CEO perspective"
- "is this ambitious enough?"
- Evaluating feature ambition and business impact

## CEO Review Framework

### 1. Vision Alignment

- Does this align with the company's long-term vision?
- Is this a 10x opportunity or a 10% improvement?
- What's the strategic moat this creates?

### 2. Market Opportunity

- How big is the market?
- Are we going after the right segment?
- What's the timing? (too early / right time / too late)

### 3. Competitive Advantage

- Why will we win?
- What's defensible about this?
- How does this compound over time?

### 4. Resource Allocation

- Is this the best use of our resources?
- What are we NOT doing by doing this?
- What's the opportunity cost?

### 5. Risk Assessment

- What's the biggest risk?
- What assumptions could be wrong?
- What's the downside scenario?

## Output Format

```markdown
## CEO Review

### Strategic Assessment
[Overall strategic fit and ambition level]

### 10x Opportunities
[Ways to make this 10x bigger/better]

### Key Risks
[Top 3 strategic risks]

### Recommendation
[Proceed / Expand / Pivot / Pass]

### Next Steps
[Specific actions to take]
```
