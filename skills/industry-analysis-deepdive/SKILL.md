# 產業分析深度技能

> 選對產業，比選對公司更重要。

## 核心目的

再好的公司也逃不過產業週期。這個技能提供五力分析、產業週期判斷、競爭格局分析，幫你判斷產業的吸引力和公司的相對位置。

---

## 產業分析框架

### 第一層：波特五力分析

| 力量 | 定義 | 判讀 | 影響 |
|------|------|------|------|
| **供應商議價力** | 供應商能否提高價格 | 強 = 成本上升，弱 = 成本穩定 | ⭐⭐⭐ 中等 |
| **買方議價力** | 客戶能否壓低價格 | 強 = 毛利下降，弱 = 毛利穩定 | ⭐⭐⭐⭐ 強 |
| **新進者威脅** | 新公司容易進入嗎 | 高 = 競爭加劇，低 = 護城河強 | ⭐⭐⭐⭐ 強 |
| **替代品威脅** | 有沒有替代產品 | 高 = 需求下降，低 = 需求穩定 | ⭐⭐⭐⭐ 強 |
| **同業競爭** | 競爭對手多不多 | 多 = 競爭激烈，少 = 競爭溫和 | ⭐⭐⭐⭐⭐ 最強 |

### 第二層：產業週期判斷

```
產業週期 = 導入期 → 成長期 → 成熟期 → 衰退期

導入期：
- 特徵：新產品、市場小、競爭少
- 機會：高成長、高風險
- 例子：AI 晶片（2023-2024）

成長期：
- 特徵：市場快速擴大、競爭開始增加
- 機會：高成長、風險中等
- 例子：AI 伺服器（2024-2025）

成熟期：
- 特徵：市場增長放緩、競爭激烈
- 機會：穩定現金流、風險低
- 例子：智慧型手機（2015-2020）

衰退期：
- 特徵：市場萎縮、競爭激烈、利潤下降
- 機會：低、風險高
- 例子：功能型手機（2010-2015）
```

### 第三層：競爭格局分析

| 格局 | 特徵 | 對公司影響 | 例子 |
|------|------|----------|------|
| **寡頭壟斷** | 少數大公司主導 | 利潤高、競爭溫和 | 台積電（晶圓代工） |
| **壟斷競爭** | 多家公司但有差異化 | 利潤中等、競爭中等 | 智慧型手機 |
| **完全競爭** | 多家公司、產品同質 | 利潤低、競爭激烈 | 代工業（鴻海等） |
| **壟斷** | 一家公司主導 | 利潤最高、競爭最少 | 罕見 |

### 第四層：新進者威脅評估

```
新進者威脅高的產業特徵：
- 進入門檻低（資本需求少）
- 技術門檻低（容易複製）
- 品牌忠誠度低（客戶容易轉換）
- 規模經濟不明顯（小公司也能活）
→ 競爭激烈，利潤下降

新進者威脅低的產業特徵：
- 進入門檻高（資本需求大）
- 技術門檻高（難以複製）
- 品牌忠誠度高（客戶粘性強）
- 規模經濟明顯（大公司有優勢）
→ 競爭溫和，利潤穩定
```

### 第五層：替代品威脅評估

```
替代品威脅高的產業：
- 有明顯的替代產品
- 替代品性能相當或更好
- 替代品價格更便宜
→ 需求下降，利潤下降

替代品威脅低的產業：
- 沒有明顯的替代產品
- 或替代品性能差
- 或替代品價格更貴
→ 需求穩定，利潤穩定
```

## 執行流程

### 步驟 1：判斷產業週期
- 導入期 / 成長期 / 成熟期 / 衰退期
- 市場規模和增長率
- 競爭對手數量和強度

### 步驟 2：分析五力
- 供應商議價力（強/中/弱）
- 買方議價力（強/中/弱）
- 新進者威脅（高/中/低）
- 替代品威脅（高/中/低）
- 同業競爭（激烈/中等/溫和）

### 步驟 3：判斷競爭格局
- 寡頭壟斷 / 壟斷競爭 / 完全競爭
- 市場集中度
- 領導者地位

### 步驟 4：評估進入門檻
- 資本門檻
- 技術門檻
- 品牌門檻
- 規模經濟

### 步驟 5：評估替代品威脅
- 有沒有替代產品
- 替代品性能對比
- 替代品價格對比

### 步驟 6：判斷產業吸引力
- 高吸引力：成長期、競爭溫和、進入門檻高
- 中等吸引力：成熟期、競爭中等、進入門檻中等
- 低吸引力：衰退期、競爭激烈、進入門檻低

### 步驟 7：判斷公司相對位置
- 領導者 / 追隨者 / 邊緣者
- 市場佔有率
- 競爭優勢

## 警示信號

🔴 **高風險產業**：
- 衰退期產業（需求下降）
- 完全競爭格局（利潤低）
- 新進者威脅高（競爭加劇）
- 替代品威脅高（需求轉移）
- 進入門檻低（競爭激烈）

🟡 **中等風險產業**：
- 成熟期產業（增長放緩）
- 壟斷競爭格局（競爭中等）
- 新進者威脅中等
- 進入門檻中等

🟢 **高吸引力產業**：
- 成長期產業（需求上升）
- 寡頭壟斷格局（競爭溫和）
- 新進者威脅低（進入門檻高）
- 替代品威脅低
- 進入門檻高（護城河強）

## 使用場景

1. **選擇產業**：優先選擇成長期、寡頭壟斷的產業
2. **評估公司**：同樣的公司，在不同產業的價值不同
3. **風險預警**：產業進入衰退期 = 可能要減碼
4. **機會判斷**：產業進入成長期 = 可能要加碼

---

## 常見誤區

❌ **誤區 1**：只看公司，不看產業
✅ **正確**：產業週期決定公司的天花板。

❌ **誤區 2**：成熟期產業就不值得投資
✅ **正確**：成熟期產業如果是寡頭壟斷，利潤反而很穩定。

❌ **誤區 3**：新進者威脅高就一定不買
✅ **正確**：要看公司有沒有護城河。護城河強的公司即使新進者威脅高也值得買。

❌ **誤區 4**：替代品威脅高就一定不買
✅ **正確**：要看替代品的成熟度。如果替代品還不成熟，威脅不大。
