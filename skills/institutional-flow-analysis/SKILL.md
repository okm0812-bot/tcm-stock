# 法人動向分析技能

> 跟著聰明錢走，往往沒錯。

## 核心目的

法人（外資、投信、自營商）的買賣動向往往領先散戶 1-2 週。這個技能幫你追蹤三大法人的資金流向，判斷大盤和個股的短期方向。

---

## 三大法人分析框架

### 第一層：三大法人定義

| 法人 | 特徵 | 操作風格 | 影響力 |
|------|------|---------|--------|
| **外資** | 國際投資機構 | 長期持有 + 短期套利 | ⭐⭐⭐⭐⭐ 最強 |
| **投信** | 國內基金公司 | 跟風 + 季度調整 | ⭐⭐⭐⭐ 強 |
| **自營商** | 券商自有資金 | 短期交易 + 對沖 | ⭐⭐⭐ 中等 |

### 第二層：關鍵指標

| 指標 | 計算 | 判讀 |
|------|------|------|
| **買賣超金額** | 買進金額 - 賣出金額 | 正 = 淨買，負 = 淨賣 |
| **累計買賣超** | 過去 N 天的買賣超總和 | 判斷中期趨勢 |
| **持倉比例** | 法人持股 / 總股數 | 判斷法人重視度 |
| **進出場訊號** | 買賣超從負轉正/正轉負 | 判斷轉折點 |

### 第三層：資金流向判斷

```
外資買超 + 投信買超 + 自營商買超 = 三大法人齊買 → 強勢訊號 🟢
外資買超 + 投信賣超 + 自營商賣超 = 外資獨買 → 中性訊號 🟡
外資賣超 + 投信賣超 + 自營商賣超 = 三大法人齊賣 → 弱勢訊號 🔴
```

---

## 執行流程

### 步驟 1：收集法人數據
- 當日三大法人買賣超金額
- 過去 5 日、10 日、20 日累計買賣超
- 各法人持倉比例
- 個股法人持股明細

### 步驟 2：分析資金流向
- 三大法人是否同向？
- 哪個法人最活躍？
- 買賣超金額是否創新高/新低？

### 步驟 3：判斷轉折點
- 買賣超從負轉正 = 可能底部
- 買賣超從正轉負 = 可能頂部
- 累計買賣超創新高 = 強勢延續

### 步驟 4：對比個股
- 個股法人持股比例
- 個股法人買賣超
- 法人持股變化趨勢

### 步驟 5：得出結論
- 大盤資金流向評估
- 個股資金流向評估
- 短期方向判斷

## 警示信號

🔴 **高風險信號**：
- 三大法人連續 3 日齊賣
- 外資持股比例創新低
- 累計買賣超從正轉負

🟡 **中等風險信號**：
- 外資賣超但投信買超（分歧）
- 買賣超金額萎縮
- 持股比例下降

🟢 **機會信號**：
- 三大法人連續 3 日齊買
- 買賣超從負轉正
- 外資持股比例創新高

## 使用場景

1. **大盤方向判斷**：三大法人齊買 = 大盤可能上漲
2. **個股進場時機**：外資買超 + 技術面確認 = 進場信號
3. **風險預警**：三大法人齊賣 = 可能要減碼
4. **短期交易**：法人動向往往領先 1-2 週

---

## 常見誤區

❌ **誤區 1**：法人買超就一定要跟著買
✅ **正確**：要看是哪個法人、買超金額多少、技術面是否確認。

❌ **誤區 2**：外資賣超就要全部賣出
✅ **正確**：外資可能只是調整持倉，不一定是看空。

❌ **誤區 3**：只看當日買賣超
✅ **正確**：要看累計買賣超趨勢，判斷中期方向。
