# Macro Monitor - 宏观经济监控

## 功能

追踪以下关键指标：
- Fed 利率决议
- CPI（消费者物价指数）
- GDP 增长率
- 非农就业数据
- 央行货币政策

## 使用方法

直接询问：
- "最新 CPI 数据"
- "Fed 什么时候降息"
- "今天有什么重要经济数据"
- "执行宏观数据监控"

## 数据源

- Trading Economics
- FRED（美联储经济数据）
- 各国央行官网
- 财联社、华尔街见闻

## 输出格式

```
【宏观经济数据】📊

🌍 国际数据
- [指标名称] [数值] [预期] [前值]
  💡 [科普解释]

🇨🇳 国内数据  
- [指标名称] [数值] [预期] [前值]
  💡 [科普解释]

📜 政策动态
- [政策标题] - [说明]
```
