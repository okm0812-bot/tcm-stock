# 全面總經數據分析技能

> 個股再好，也逃不過大盤。

## 核心目的

Fed 利率和 CPI 只是基礎。這個技能提供 PMI、失業率、消費者信心、全球經濟環境等全面總經指標，幫你判斷大盤方向。

---

## 全面總經分析框架

### 第一層：美國經濟指標

| 指標 | 發布頻率 | 判讀 | 對股市影響 |
|------|---------|------|----------|
| **Fed 利率** | 6 週一次 | 升息 = 利空，降息 = 利多 | ⭐⭐⭐⭐⭐ 最強 |
| **CPI（消費者物價）** | 月度 | > 3% = 通膨高，< 2% = 通膨低 | ⭐⭐⭐⭐ 強 |
| **PMI（採購經理人指數）** | 月度 | > 50 = 擴張，< 50 = 收縮 | ⭐⭐⭐⭐ 強 |
| **失業率** | 月度 | < 4% = 低失業，> 5% = 高失業 | ⭐⭐⭐ 中等 |
| **消費者信心指數** | 月度 | > 100 = 信心高，< 90 = 信心低 | ⭐⭐⭐ 中等 |
| **非農就業人數** | 月度 | > 20 萬 = 就業好，< 10 萬 = 就業差 | ⭐⭐⭐ 中等 |
| **GDP 成長率** | 季度 | > 2% = 成長好，< 1% = 成長差 | ⭐⭐⭐⭐ 強 |
| **美債殖利率** | 即時 | 上升 = 利空，下降 = 利多 | ⭐⭐⭐⭐ 強 |

### 第四層：總經景氣判斷

```
強勢景氣：
- Fed 降息 + PMI > 50 + 失業率低 + 消費者信心高
→ 股市通常上漲 🟢

弱勢景氣：
- Fed 升息 + PMI < 50 + 失業率高 + 消費者信心低
→ 股市通常下跌 🔴

轉折點：
- PMI 從 > 50 跌到 < 50 = 可能頂部
- PMI 從 < 50 升到 > 50 = 可能底部
- Fed 從升息轉降息 = 可能底部
```

## 警示信號

🔴 **高風險信號**：
- PMI 從 > 50 跌到 < 50（景氣轉弱）
- Fed 升息 + CPI 高企（滯脹風險）
- 失業率上升 + 消費者信心下降
- 地緣政治風險升高

🟡 **中等風險信號**：
- PMI 在 50 附近波動（景氣不確定）
- Fed 利率政策不明確
- 出口數據疲弱

🟢 **機會信號**：
- PMI 從 < 50 升到 > 50（景氣轉強）
- Fed 從升息轉降息（流動性改善）
- 失業率低 + 消費者信心高
- 出口數據強勁

---

## 使用場景

1. **大盤方向判斷**：總經數據決定大盤方向
2. **產業輪動**：景氣強時選科技，景氣弱時選防守
3. **進場時機**：景氣轉折點是最好的進場時機
4. **風險預警**：總經惡化 = 可能要減碼

---

## 常見誤區

❌ **誤區 1**：只看 Fed 利率
✅ **正確**：要看 Fed 利率 + PMI + 失業率 + 消費者信心的組合。

❌ **誤區 2**：CPI 高就一定要賣股票
✅ **正確**：要看 Fed 是否升息。升息才是利空。

❌ **誤區 3**：中國經濟差就不買台股
✅ **正確**：要看台灣出口是否受影響。有些產業不受影響。

❌ **誤區 4**：只看美國經濟
✅ **正確**：要看美國 + 中國 + 全球 + 台灣的組合。
