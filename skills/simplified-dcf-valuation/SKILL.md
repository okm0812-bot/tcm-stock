# 簡化版 DCF 估值技能

> DCF 不是火箭科學，只是「預測未來現金流 + 折現」。

## 核心目的

DCF（折現現金流）是巴菲特最愛的估值方法。但大多數人被複雜公式嚇跑了。這個技能提供**簡化版 DCF**，讓你 5 分鐘內估算一家公司的內在價值。

---

## 簡化版 DCF 公式

```
內在價值 = Σ(未來5年 FCF / (1+折現率)^年數) + 終值 / (1+折現率)^5

簡化版：
1. 預測未來 5 年 FCF（年增長率 x%）
2. 用折現率 10% 折現
3. 加上終值（第 5 年 FCF × 永續成長率 / (折現率 - 永續成長率)）
4. 除以股數 = 每股內在價值
```

## 三步驟快速 DCF

### 步驟 1：收集基礎數據

| 數據 | 說明 | 來源 |
|------|------|------|
| **最新 FCF** | 最近一年的自由現金流 | 財報 |
| **股數** | 流通股數 | 財報 |
| **預期成長率** | 未來 5 年 FCF 年增長率 % | 分析師預測 |
| **折現率** | 通常用 10%（保守） | 固定值 |
| **永續成長率** | 第 5 年後的永續增長率，通常 2-3% | 固定值 |

### 步驟 2：預測 5 年 FCF

```
假設：
- 2025年 FCF：100 億
- 預期成長率：15%（未來 5 年）

預測：
- 2026年 FCF = 100 × 1.15 = 115 億
- 2027年 FCF = 115 × 1.15 = 132 億
- 2028年 FCF = 132 × 1.15 = 152 億
- 2029年 FCF = 152 × 1.15 = 175 億
- 2030年 FCF = 175 × 1.15 = 201 億
```

### 步驟 3：折現 + 計算內在價值

```
折現率 = 10%

2026年 FCF 現值 = 115 / 1.10^1 = 104.5 億
2027年 FCF 現值 = 132 / 1.10^2 = 109.1 億
2028年 FCF 現值 = 152 / 1.10^3 = 114.2 億
2029年 FCF 現值 = 175 / 1.10^4 = 119.6 億
2030年 FCF 現值 = 201 / 1.10^5 = 124.8 億

5年 FCF 現值合計 = 572.2 億

終值 = 2030年 FCF × (1 + 永續成長率) / (折現率 - 永續成長率)
     = 201 × 1.03 / (0.10 - 0.03)
     = 207 / 0.07
     = 2,957 億

終值現值 = 2,957 / 1.10^5 = 1,835 億

企業價值 = 572.2 + 1,835 = 2,407 億

每股內在價值 = 2,407 / 股數
```

## 參數設定指南

### 預期成長率（5年）

| 情景 | 成長率 | 適用公司 |
|------|--------|---------|
| 高成長 | 15-20% | AI、新興產業 |
| 中等成長 | 8-12% | 傳統製造、科技 |
| 低成長 | 3-5% | 成熟產業、公用事業 |
| 衰退 | -5-0% | 夕陽產業 |

### 折現率

| 情景 | 折現率 | 說明 |
|------|--------|------|
| 保守 | 12% | 高風險公司 |
| 中等 | 10% | 一般公司（推薦） |
| 樂觀 | 8% | 低風險公司 |

### 永續成長率

| 情景 | 永續成長率 | 說明 |
|------|-----------|------|
| 保守 | 2% | 低於 GDP 成長率 |
| 中等 | 3% | 等於 GDP 成長率 |
| 樂觀 | 4% | 高於 GDP 成長率 |

## 敏感性分析

DCF 對參數很敏感，建議做「敏感性分析」：

```
內在價值 = f(成長率, 折現率, 永續成長率)

製作表格：
- 橫軸：成長率（10%, 12%, 15%, 18%, 20%）
- 縱軸：折現率（8%, 10%, 12%）
- 格子：對應的內在價值

這樣可以看出內在價值的波動範圍。
```

## 使用場景

1. **新股票分析**：快速估算合理價位
2. **估值檢視**：定期重新估值，看是否還便宜
3. **加碼決策**：股價跌到內在價值 70% 以下時考慮加碼
4. **止損判斷**：如果基本面惡化，重新估值後發現內在價值下降，考慮止損

## 常見誤區

❌ **誤區 1**：DCF 結果就是準確的內在價值
✅ **正確**：DCF 只是估值範圍，不是精確值。應該用 70-130% 的內在價值作為買賣區間。

❌ **誤區 2**：用歷史成長率預測未來
✅ **正確**：歷史成長率只是參考，要考慮產業週期、競爭格局變化。

❌ **誤區 3**：折現率越低越好
✅ **正確**：折現率應該反映風險。高風險公司應該用高折現率。

❌ **誤區 4**：一次性計算就夠了
✅ **正確**：應該定期重新估值（每季或每年），因為基本面會變化。
