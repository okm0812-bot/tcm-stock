# 技術面與風險量化分析技能

> 基本面告訴你買什麼，技術面告訴你什麼時候買。

## 核心目的

基本面分析告訴你一家公司值多少錢，但技術面分析幫你找到最好的進場時機。這個技能結合技術面訊號和風險量化，幫你做出更精準的投資決策。

---

## 技術面分析框架

### 第一層：基礎技術指標

| 指標 | 計算 | 判讀 |
|------|------|------|
| **RSI（相對強弱指數）** | 14日 RSI | > 70 超買，< 30 超賣 |
| **MACD** | 12日 EMA - 26日 EMA | 金叉買，死叉賣 |
| **布林帶** | 中線 ± 2 標準差 | 上軌壓力，下軌支撐 |
| **移動平均線** | 20日、50日、200日 | 價格在均線上方 = 上升趨勢 |

### 第二層：支撐壓力位

```
支撐位 = 股價難以跌破的價位
壓力位 = 股價難以突破的價位

判斷方法：
1. 歷史低點 = 支撐位
2. 歷史高點 = 壓力位
3. 整數位（如 200、250）= 心理位
4. 移動平均線 = 動態支撐/壓力
```

### 第三層：進場訊號

| 訊號 | 說明 | 可靠度 |
|------|------|--------|
| **MACD 金叉** | 快線穿過慢線 | ⭐⭐⭐ 中等 |
| **RSI 底背離** | 股價創新低但 RSI 不創新低 | ⭐⭐⭐⭐ 較高 |
| **布林帶反彈** | 股價觸及下軌後反彈 | ⭐⭐⭐ 中等 |
| **支撐位反彈** | 股價在支撐位反彈 | ⭐⭐⭐ 中等 |

## 風險量化框架

### 第一層：風險指標

| 指標 | 計算 | 說明 |
|------|------|------|
| **波動率（Volatility）** | 股價日漲跌幅的標準差 | 波動率高 = 風險高 |
| **最大回撤（MDD）** | 從高點到低點的最大跌幅 % | 判斷最壞情況 |
| **VaR（風險價值）** | 在 95% 信心下，最大虧損 % | 量化風險 |
| **夏普比率** | (報酬率 - 無風險利率) / 波動率 | 風險調整後的報酬 |

### 第二層：簡化版風險評估

```
最壞情況 = 當前股價 × (1 - 最大回撤%)
中等情況 = 當前股價 × (1 - 平均回撤%)
最好情況 = 當前股價 × (1 + 目標漲幅%)

風險/報酬比 = (最好情況 - 當前股價) / (當前股價 - 最壞情況)
```

### 第三層：風險等級

| 等級 | 波動率 | 最大回撤 | 評價 |
|------|--------|---------|------|
| 🟢 低風險 | < 15% | < 20% | 適合保守投資者 |
| 🟡 中風險 | 15-25% | 20-40% | 適合一般投資者 |
| 🔴 高風險 | > 25% | > 40% | 適合激進投資者 |

## 執行流程

### 步驟 1：收集技術數據
- 過去 1 年的日線數據
- 計算 RSI、MACD、布林帶
- 標記支撐壓力位

### 步驟 2：分析技術訊號
- 當前 RSI 位置（超買/超賣/中性）
- MACD 是否金叉/死叉
- 股價是否在布林帶上軌/中線/下軌

### 步驟 3：計算風險指標
- 波動率
- 最大回撤
- VaR（簡化版）

### 步驟 4：確定進場區間
- 支撐位 = 進場區間下限
- 壓力位 = 進場區間上限
- 最好的進場點 = 支撐位 + 技術訊號確認

### 步驟 5：設定止損止利
- 止損 = 支撐位下方 5%
- 止利 = 壓力位上方 5%

## 使用場景

1. **進場時機**：基本面好 + 技術面訊號確認 = 最佳進場點
2. **止損設定**：支撐位破位 = 止損信號
3. **加碼時機**：股價回到支撐位 + RSI 超賣 = 加碼機會
4. **風險控制**：風險/報酬比 < 1 = 不進場

---

## 常見誤區

❌ **誤區 1**：只看技術面，忽視基本面
✅ **正確**：技術面 + 基本面結合，才是最強組合。

❌ **誤區 2**：RSI > 70 就一定要賣
✅ **正確**：RSI 只是參考，強勢股可以長期在 70 以上。

❌ **誤區 3**：支撐位一定會反彈
✅ **正確**：支撐位破位後，會變成新的壓力位。

❌ **誤區 4**：風險/報酬比 > 1 就一定要買
✅ **正確**：還要考慮成功概率。風險/報酬比 > 1 但成功率 < 30% 也不值得。
